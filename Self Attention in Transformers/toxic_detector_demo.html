How Self-Attention Powers This

1
Tokenize & Embed Each word → integer ID → 16-dimensional embedding vector from a learned table E.
2
Compute Q, K, V Each embedding is projected into three vectors: Query (what am I looking for?), Key (what do I offer?), Value (what info do I carry?).
3
Attention Scores Scores = Q @ Kᵀ / √d_k. High score → word i attends strongly to word j. Softmax normalizes to probabilities.
4
Weighted Sum → Pool → Classify Output = Softmax(scores) @ V. Mean-pooled across tokens → Linear → Sigmoid → P(toxic).
5
Trained via Backpropagation Weight matrices Wq, Wk, Wv and embeddings are updated each step to minimize Binary Cross-Entropy loss.
